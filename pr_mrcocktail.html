Mister Cocktail

Mister Cocktail is a Ruby on Rails Application for all cocktail lovers around the world!

This project was initially built within the scope of the Bootcamp at Le Wagon with Ruby, HTML, SCSS and JavaScript.

Later on, i've spent time optimizing the database seeding, search and filters as well as creating an overall enjoyable user experience.

Goal and Process

The purpose of this project is to allow users to search, filter and create their own cocktails.
The design process had 3 major phases:

Phase 1 - Search

For the database seeding, i parsed data from the test API key avaliable at the cocktail db. Futhermore, i used PG Search to allow the user to search for a specific cocktail or ingredient.

Phase 2 - Filter

On the cocktail's model I've created 3 collections for category, alcoholic and glass. The filters, located on the navigation bar, were easily set up using dropown menus and simple form.

Phase 3 - Create

CRUD methods were implemented on the cocktail's and dose's controllers for user interaction. Regarding the hosting and uploading of images, i used the Cloudinary platform.

Challenges and solving process

Web design concept

I was looking forward to create an app that would resemble the experience of drinking a cocktail.

For the background, i customized the JavaScript function availiable at bubbly-bg, to produce the same appearance as a gasified drink.

The cocktails should look like coasters, that one gets at a bar when ordering a drink. For this I've learned how to apply text to an svg path.

The search, filter and create features are concentrated and allways availiable on the top navigation bar. For orientation purposes, each color of the buttons matches a CRUD method.

I've also worked with the Bootstrap grid for the layout in contrast to custom hover animations, in order to provide an playful user experience.
